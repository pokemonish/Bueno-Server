# Bueno Server 3000
Super secure server ever!!11

## Dependencies
Erlang

## How to start
- type ```erl``` in terminal
- type ```c(httpd).``` in terminal
- type ```{ok, Port} = httpd:start().``` in terminal (AND SERVER WILL START!!1!)
- type ```stop(Port).``` in terminal (AND SERVER WILL STOP!!1!)
